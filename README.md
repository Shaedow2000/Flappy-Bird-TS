# Flappy Bird Game Colne In TypeScript.

- It is a simple colne of the old famous game know as "flappy bird", made to be played in the web browser.

## I ) How to play it:
- Play the game using the space key from your key board, and try avoiding the green obstacles!
## II ) Points:
- All points will be saved in **LocalStorage**, but if the score is beaten, or you got a better score then the score will be replaced with the new **record**.
## III ) Updates:
- The game will be updated from week to week, or once a week depending on time factors.
